# Martin Bozhilov Portfolio

Single-page portfolio for Martin Bozhilov, built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, and Lucide React.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production Build

```bash
npm run build
npm run start
```

## Deployment

The app is ready for Vercel:

1. Push the repository to GitHub.
2. Import it in Vercel.
3. Use the default Next.js framework settings.

## Content

Portfolio content lives in `data/portfolio.ts`. Reusable UI sections live in `components/`.

## Profile Image

The page works without a photo by using a monogram identity. A future `public/profile.jpg` can be added and wired into the hero or about section.
